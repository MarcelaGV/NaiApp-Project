# Descripción

NaiApp es una aplicación web desarrollada para optimizar la administración
de salones de belleza mediante la automatización del proceso de agendamiento.

El sistema permite:

- Gestionar citas y reservas
- Administrar clientes y servicios
- Llevar control financiero básico
- Mejorar la organización del negocio
- Optimizar la experiencia del cliente

---

# Objetivo

> Desarrollar una aplicación web que permita automatizar el proceso de
> agendamiento para los clientes en un salón de belleza y que permita
> a los administradores monitorear y gestionar clientes, servicios y
> finanzas de manera organizada y eficiente.
